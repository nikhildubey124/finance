INTEGRATION OPTIONS:
--------------------

Option 1: Use Base Template (Recommended)
------------------------------------------
Extend the base template for new pages:

{% extends "base.html" %}
{% block content %}
    <!-- Your content -->
{% endblock %}

Option 2: Add Includes to Existing Templates
---------------------------------------------
Add these three includes to your existing templates:

In <head>:
{% include 'includes/loader_assets.html' %}

After <body>:
{% include 'includes/loader_html.html' %}

Before </body>:
{% include 'includes/loader_scripts.html' %}

Option 3: Manual Integration
-----------------------------
Copy the loader HTML, CSS, and JS directly into templates.
See LOADER_IMPLEMENTATION_EXAMPLE.html for a complete example.

USAGE EXAMPLES:
---------------

1. Automatic (No Code Needed):
   <form method="POST">
       <button type="submit">Submit</button>
   </form>
   <!-- Loader shows automatically! -->

2. Custom Message:
   <form data-loading-message="Saving data..."
         data-loading-subtext="Please wait">
   </form>

3. Disable Loader:
   <form class="no-loader">
       <!-- Won't trigger loader -->
   </form>

4. Manual Control:
   GlobalLoader.show('Processing...', 'Please wait');
   // ... do something ...
   GlobalLoader.hide();

CONFIGURATION:
--------------

To customize behavior, edit static/loader.js:

CONFIG = {
    minDisplayTime: 300,        // Min time to show (ms)
    autoHideTimeout: 30000,     // Auto-hide after 30s
    showOnNavigation: true,     // Show on link clicks
    showOnFormSubmit: true,     // Show on form submit
    showOnPageLoad: true        // Show during page load
};

TROUBLESHOOTING:
----------------

Q: Loader not showing?
A: Check that includes are added in correct order
   (CSS in head, HTML after body, JS before other scripts)

Q: Loader not hiding?
A: Check browser console for JavaScript errors
   Failsafe will auto-hide after 30 seconds

Q: Loader showing for quick actions?
A: Add class="no-loader" to those elements

Q: Want different colors?
A: Edit static/loader.css color values
